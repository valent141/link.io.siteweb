Présentation

Link.IO est une API (disponible pour C# et Java) qui permet aux développeurs d'applications l'utilisant de s’abstraire des problématiques de communication entre périphériques. Link.IO offre un socle stable d'outils pour vos échanges de données (documents, images, textes...), et vous permet de vous concentrer sur votre cœur de métier !



Gestion événementielle

Échangez vos données via des événements sur lesquels les applications sont en écoute.



Gestion des identités

Chaque utilisateur peut être identifié à l'intérieur d'un groupe pour une application donnée.



Sécurité des échanges

Un groupe possède des droits administrables qui sont vérifiés à chaque envoi d'événement d'un de ses membres pour plus de sécurité.



Gestion des communications cross-platform

La communication entre différentes plateformes est totalement transparente. Vos données peuvent transiter de l'API C# vers l'API Java et inversement !

Principe

Architecture

Link.IO utilise les WebSockets pour la gestion des communications entre périphériques avec un serveur qui s'occupe du relais des événements aux bons destinataires.

Il est possible d'administrer et de monitorer les événements pendant le développement et la production de votre application grâce à une interface !
