Share data between your Web, Windows and Android apps!
With Link.IO, program and synchronize your event exchanges between your apps as simply for you as possible!

Presentation

Link.IO is an API (available for C# and Java languages) which allows app developers using it to cut them off from communication issues between devices. Link.IO offers a steady base of tools for your data exchanges (documents, images, texts...), and allows you to concentrate on your core businesses!



Event management

Share data through events on which apps are listening.



Identity management

Each user can be authenticated in a group for a given app.



Exchange safety

A group has administrable rights which are checked every time an event is sent by one of its member to improve safety.



Cross-platform communications management

Communication between different platforms is totally unnoticeable. Data can pass through C# API towards Java API and vice versa!

Architecture

Link.IO uses WebSockets to manage communications between devices with a server which relays events to the right recipients.

You can manage and monitor events during the development and the production of your app thanks to an interface!

Demo

The above area lets you draw in collaboration with other persons currently visiting our website. Each stroke you are drawing is sent to others and vice versa.
You can check for yourself by opening an other demo from the same device or from another one.
